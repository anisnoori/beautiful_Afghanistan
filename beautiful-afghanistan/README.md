# **Beautiful Afghanistan**

A modern tourism website showcasing the beauty, culture, history, and cities of Afghanistan.

This project presents Afghanistan through interactive city pages, image galleries, and a cinematic landing experience designed with modern web design principles.

## 

## **Live Website**

[https://anisnoori.github.io/BeautifulAfghanistan/](https://anisnoori.github.io/BeautifulAfghanistan/)

## **Project Goal**

The purpose of this project is to:

* Present Afghanistan as a beautiful tourism destination  
* Highlight historical cities and cultural identity  
* Practice modern frontend development  
* Build a professional portfolio-level website

## **Features**

Cinematic Hero Section (Apple / Netflix style entrance)  
Smooth animations and transitions  
City exploration pages  
Image galleries for each city  
Search cities functionality  
Responsive design  
Clean UI layout  
GitHub Pages deployment

## **Cities Included**

## **How to Run Locally**

1. Clone repository

git clone https://github.com/anisnoori/BeautifulAfghanistan.git

2. Open folder

cd BeautifulAfghanistan

3. Open `index.html` in browser.
